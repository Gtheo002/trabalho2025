Uma calculadora de linha de comando para números complexos feita em Java.
Ela permite:

Fazer operações com complexos: +, -, *, /, ^
Usar funções como sqrt, root, conj, exp, ln e abs
Usar variáveis dentro das expressões
Ver a expressão em formato de árvore (LISP)
Testar se duas expressões são equivalentes
Aceita entradas como:
3+4i, (1+2i)*(3-i), sqrt(2+3i), a + i, etc.
